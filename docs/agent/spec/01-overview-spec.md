> 作成日: 2026-03-30  
> バージョン: 1.0.0

---

## 1. プロジェクト概要

| 項目 | 内容 |
|------|------|
| プロジェクト名 | Web Quizize |
| 目的 | URL を貼り付けるだけで AI が自動的に穴埋め・4択クイズを生成し、Webコンテンツの内容理解を助ける学習ツール |
| ターゲット | Webページを読んで知識を定着させたい学習者・研究者・一般ユーザー |
| AI エンジン | さくらの AI Engine（OpenAI 互換 API） |

---

## 2. 主要機能一覧

### 2.1 コア機能

| # | 機能名 | 概要 | 優先度 |
|---|--------|------|--------|
| F-01 | URL 入力・コンテンツ取得 | ユーザーが入力した URL からページのテキストコンテンツを取得する | 必須 |
| F-02 | AI クイズ自動生成 | 取得コンテンツを AI で解析し、穴埋めクイズ・4択クイズを生成する | 必須 |
| F-03 | 難易度設定 | 初級 / 中級 / 上級 の 3 段階でクイズ難易度を選択できる | 必須 |
| F-04 | クイズ回答 | 4択形式で回答し、即時正誤フィードバックと解説を表示する | 必須 |
| F-05 | 学習記録 | 正答率・学習時間・挑戦回数などの学習履歴を記録・可視化する | 必須 |
| F-06 | ユーザー認証 | メールアドレス + パスワードによるサインアップ・ログイン | 必須 |

### 2.2 拡張機能

| # | 機能名 | 概要 | 優先度 |
|---|--------|------|--------|
| F-07 | クイズ一覧・再挑戦 | 過去に生成したクイズを一覧表示し、再挑戦できる | 高 |
| F-08 | 進捗プログレスバー | クイズ回答中の進捗（何問目 / 全問）をリアルタイム表示 | 高 |
| F-09 | 苦手ジャンル分析 | 正答率の低い分野を可視化しユーザーに通知する | 中 |
| F-10 | クイズ削除 | 不要なクイズをダッシュボードから削除できる | 中 |

## 5. ユーザーフロー

```
[未認証ユーザー]
  ランディングページ (/)
    ├── URL入力 → 「無料で試す」→ サインアップ画面へ
    ├── ログインリンク → ログイン画面へ
    └── サインアップ → ログイン画面へ

[認証済みユーザー]
  ダッシュボード (/dashboard)
    ├── URL入力 → クイズ生成中 → クイズ回答画面
    ├── 過去クイズ一覧 → クイズ回答画面（再挑戦）
    └── 学習統計の確認
  
  クイズ回答画面 (/quiz/:id)
    └── 全問回答 → 結果画面 (/quiz/:id/result)
```

## 6. クイズ生成フロー詳細

```
1. ユーザーが URL を入力し「クイズ生成」を送信
2. サーバーが URL からページコンテンツを fetch
3. HTML をパースしてテキストを抽出（メタ情報・見出し・本文）
4. さくらの AI Engine へコンテンツ + 難易度を送信
5. AI がキーワードを抽出し、穴埋め問題・4択選択肢を生成
6. DB に Quiz / QuizQuestion / QuizChoice を保存
7. フロントエンドにクイズIDを返却
8. クイズ回答画面へリダイレクト
```

## 7. 非機能要件

| 項目 | 要件 |
|------|------|
| クイズ生成時間 | URL 送信から生成完了まで 10 秒以内を目標 |
| クイズ問題数 | 1クイズあたり 5〜10 問（デフォルト 5 問） |
| 対応 URL | 公開アクセス可能な HTTP/HTTPS の Webページ |
| 認証 | メール + パスワード認証（better-auth） |
| セキュリティ | SSRF 対策（プライベート IP へのアクセス禁止）、XSS 対策 |
| レスポンシブ | PC / タブレット / モバイル対応 |
| エラーハンドリング | URL 取得失敗・AI API 失敗時の適切なエラーメッセージ表示 |
