## 2. 画面一覧

| 画面名 | パス | 認証 | 説明 |
|--------|------|------|------|
| ランディングページ | `/` | 不要 | URL入力フォーム・機能紹介 |

| クイズ結果 | `/quiz/$quizId/result` | 必要 | 最終スコア・解説表示 |

## 3. 画面遷移図

```
[/]
 ├── 「無料で試す」ボタン → [/signup]
 ├── 「ログイン」リンク → [/login]
 └── URL入力 + 「クイズ生成」ボタン → [/login] (未認証の場合リダイレクト)

[/signup] ──登録完了──→ [/login]
[/login] ──ログイン成功──→ [/dashboard]

[/dashboard]
 ├── URL入力 + 「クイズ生成」→ [/quiz/generating] ──生成完了──→ [/quiz/$quizId]
 ├── 過去クイズ「挑戦する」→ [/quiz/$quizId]
 └── クイズ「削除」→ 確認ダイアログ → 削除実行

[/quiz/generating]
 └── 生成完了（oRPC polling or redirect）→ [/quiz/$quizId]

[/quiz/$quizId]
 └── 全問回答完了 → [/quiz/$quizId/result]

[/quiz/$quizId/result]
 ├── 「もう一度挑戦」→ [/quiz/$quizId] (新しいAttemptで再開)
 └── 「ダッシュボードへ」→ [/dashboard]
```

## 4. 各画面の詳細設計

### 4.1 ランディングページ (`/`)

#### レイアウト構成

```
┌──────────────────────────────────────────┐
│ NAV: ロゴ | 使い方 | 機能 | 料金 | [無料で試す] │
├──────────────────────────────────────────┤
│ HERO                                      │
│  "URLを貼るだけで クイズになる"           │
│  ┌────────────────────────┬──────────┐   │
│  │ URL入力フォーム         │クイズ生成→│   │
│  └────────────────────────┴──────────┘   │
│  [3秒] [12+] [98%] 統計バッジ             │
├──────────────────────────────────────────┤
│ HOW IT WORKS: 3ステップ (グリッド)        │
├──────────────────────────────────────────┤
│ DEMO: 元記事 → 生成クイズ のビジュアル    │
├──────────────────────────────────────────┤
│ FEATURES: 4カード (文脈AI/3秒/難易度/記録) │
├──────────────────────────────────────────┤
│ CTA: "さあ、解いてみよう"                 │
├──────────────────────────────────────────┤
│ FOOTER                                   │
└──────────────────────────────────────────┘
```

#### 主要コンポーネント

- `HeroSection`: URL入力フォーム + CTAボタン
- `HowItWorksSection`: 3ステップカード
- `DemoSection`: 入力/出力の並列表示
- `FeaturesSection`: 機能カード4枚
- `CtaSection`: 最終CTA

#### インタラクション

- URL入力フォームで URL を入力し「クイズ生成 →」ボタン押下
  - 未ログイン → `/login?redirect=/quiz/generate&url=<encoded_url>` へリダイレクト
  - ログイン済み → クイズ生成 API を呼び出し `/quiz/generating` へ遷移
- スクロール時に `.reveal` クラスで要素をフェードインアニメーション

#### 状態管理

```typescript
type QuizState = {
  currentQuestionIndex: number;
  answers: Map<questionId, { choiceId: string; isCorrect: boolean }>;
  showExplanation: boolean;
  attemptId: string;

```

#### インタラクション

| アクション | 結果 |
|-----------|------|
| 選択肢をクリック | `quizAttempt.answer` を呼び出し → 正誤フィードバック表示 |

| 不正解の場合 | 選択肢ボーダーを `--accent2` (赤ピンク) に変更、正解を強調 |
| 「次の問題」クリック | 次問題へ、または最終問なら `quizAttempt.complete` → 結果画面へ |

## 5. ルーター定義（TanStack Router）

```typescript
// routes/__root.tsx    — 共通レイアウト (ヘッダー・フッター)
// routes/index.tsx     — ランディングページ (/)
// routes/login.tsx     — ログイン (/login)
// routes/signup.tsx    — サインアップ (/signup)  ← 新規
// routes/dashboard.tsx — ダッシュボード (/dashboard)
// routes/quiz/
//   generating.tsx     — 生成中 (/quiz/generating)  ← 新規
//   $quizId/
//     index.tsx        — 回答画面 (/quiz/$quizId)    ← 新規
//     result.tsx       — 結果画面 (/quiz/$quizId/result) ← 新規
```

## 6. 認証ガード

TanStack Router の `beforeLoad` を使い、`/dashboard` 以下への未認証アクセスをリダイレクト。

```typescript
// routes/dashboard.tsx
export const Route = createFileRoute('/dashboard')({
  beforeLoad: async ({ context }) => {
    if (!context.session) {
      throw redirect({ to: '/login' });
    }
  },
});
```

## 7. レスポンシブ対応方針

| ブレークポイント | 対応内容 |
|----------------|---------|
| `sm` (640px〜) | モバイル最適化（シングルカラム） |
| `md` (768px〜) | タブレット（2カラムレイアウト） |
| `lg` (1024px〜) | PC（フルレイアウト） |

- ランディングページのステップグリッドはモバイルで 1 カラムに変更
- ダッシュボードのカードはモバイルで 2×2 → 1 カラムに変更
- クイズ選択肢は 2×2 グリッド（モバイルでも維持）
